# Скрипт для конвертации SVG файлов в React компоненты

Перед использованием скрипта, необходимо положить все .svg файлы в папку `scripts/transform-svg/input`.

Внутри папки `input` можно положить папки с .svg файлами, скрипт пробежится по всем директориям и отберёт файлы с нужным расширением.

После, чтобы запустить скрипт, необходимо ввести в терминале команду `scripts:icons`

---

Для работы скрипта используются библиотеки:
* [**ts-node**](https://github.com/TypeStrong/ts-node) - обёртка над nodejs с поддержкой typescript
* [**svgr**](https://github.com/gregberge/svgr) - трансформация SVG в React компоненты
